# Vertex

The operating system for developers.

Vertex is a modern learning platform that helps developers learn, practice, build real projects, prepare for interviews, and become job-ready.

## Status

In active development.

## Core Idea

Vertex is not just a course platform. It is a developer growth system that combines learning, coding practice, AI feedback, progress tracking, project building, and interview preparation.

## Project Principles

- Build for the future, ship in small steps.
- Security is a feature.
- Every feature must solve a real problem.
- Documentation is part of the code.
- AI assists, humans decide.
- Understand every line.

## Tech Direction

Web-first platform with a mobile companion planned later.
